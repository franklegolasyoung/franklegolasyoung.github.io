Hexo command not found

Frank Young

2021-06-04 14:16:36

Welcome to my blog! This is the first blog I write about some issues in using Hexo. After building up this blog, I haven’t using Hexo for about two weeks. Now the hexo commands are not available.

问题：

在搭建博客后进入新终端，重新编辑使用hexo，发现hexo命令无效的问题。

排查步骤

1. 查看nodejs和npm版本

正确无误，排除二者影响。

2. 使用npm重新安装hexo

重新安装成功，并没有解决问题。

3. 检查系统环境变量

系统环境变量中并没有hexo和npm的链接项，说明第一次安装并没有配置好hexo的环境变量，导致再次访问终端后，丢失之前配置的环境，无法正常使用hexo。
